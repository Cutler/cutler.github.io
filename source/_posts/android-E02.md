# 第一节 高效的显示图片 #
　　本章将讲解一些常用的处理和加载`Bitmap`(位图)对象到应用程序中的技巧，这些技术可以在某种程度上保持你的`UI组件`能够及时响应操作，并且避免图片超出应用的内存限制。 如果你不注意这些，那么图片就会迅速地消耗你的可利用内存，甚至于会抛出一个可怕的异常并导致应用的崩溃：
```
java.lang.OutofMemoryError: bitmap size exceeds VM budget
```
